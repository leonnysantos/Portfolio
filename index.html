Sobre Mim

Olá! Meu nome é Leonny Patrocínio santos, tenho 22 anos. Sou um desenvolvedor web dedicado e apaixonado, com foco especial no frontend. Minhas principais habilidades incluem HTML, CSS e JavaScript, e adoro usar essas ferramentas para criar sites interativos e envolventes que proporcionem ótimas experiências aos usuários. Busco sempre aprender mais e acompanhar as tendências em constante evolução do setor de desenvolvimento web, meu próximo passo é estudar o React e o TypeScript. Acredito que o aprendizado contínuo é essencial para crescer profissionalmente e oferecer soluções inovadoras aos projetos que abordo. Meu objetivo é focar no desenvolvimento frontend, pois acredito que a interface do usuário é um dos aspectos mais importantes para proporcionar uma experiência positiva aos usuários e, assim, alcançar resultados excepcionais para os clientes e suas metas.
