Sobre Mim

Olá, meu nome é Leonny, tenho 23 anos e sou um desenvolvedor Full Stack. Estou iniciando minha carreira na área de desenvolvimento web. Apesar de estar no começo da minha jornada profissional, tenho uma sólida base técnica em diversas tecnologias, adquirida através de estudos intensivos com o curso Full Stack Javascript da OneBitCode e com projetos pessoais. Tenho habilidades em HTML e CSS, que utilizo para criar estruturas semânticas e estilizar interfaces, assegurando responsividade e design atrativo. Com JavaScript e TypeScript, desenvolvo funcionalidades dinâmicas com foco na eficiência e clareza do código. Utilizo Sass e Bootstrap para melhorar a produtividade e manter o código CSS organizado. No desenvolvimento de aplicações mais complexas, uso frameworks e bibliotecas como React e Next.js para criar interfaces de usuário reativas e escaláveis. No backend, trabalho com Node.js para desenvolver servidores e APIs, além de ter conhecimento em bancos de dados como MongoDB e PostgreSQL para gerenciar dados de maneira eficiente. Também estou familiarizado com Git para controle de versão e colaboração em projetos, seguindo as melhores práticas de versionamento.
